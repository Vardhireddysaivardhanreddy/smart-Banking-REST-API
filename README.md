# Smart Banking REST API

Developed a scalable Banking REST API using Spring Boot, JPA, Hibernate, and MySQL with complete CRUD operations and RESTful architecture.

## Features
- Create Account
- Read Account
- Update Account
- Delete Account
- MySQL Database Integration
- REST API Testing with Postman

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman
- GitHub

## API Endpoints

### Create Account
POST /accounts

### Get All Accounts
GET /accounts

### Update Account
PUT /accounts/{id}

### Delete Account
DELETE /accounts/{id}

## Database
Integrated MySQL database using Hibernate and JPA.

## Testing
All APIs tested successfully using Postman.
